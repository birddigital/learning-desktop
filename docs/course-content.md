# Get Ahead of AI - Course Content Structure

> **Course Goal**: Teach students how to not fall behind in the oncoming AI onslaught and get ahead of AI instead of it being ahead of you.

## Course Overview

| Property | Value |
|----------|-------|
| **Title** | Get Ahead of AI |
| **Duration** | 6 weeks (self-paced) |
| **Level** | Beginner to Intermediate |
| **Prerequisites** | Basic computer literacy |
| **Format** | Interactive chat with AI tutor, video lessons, hands-on exercises |

## Module Structure

```
course:get-ahead-of-ai
├── module-1: reality-check
├── module-2: ai-literacy
├── module-3: your-ai-workflow
├── module-4: ai-in-your-domain
├── module-5: staying-ahead
└── module-6: human-advantage
```

## Module 1: The Reality Check

**Goal**: Confront the AI revolution honestly and assess where you stand.

### Lessons

#### 1.1: Welcome to the Acceleration
- **Concept**: Understanding exponential technological change
- **Video**: "The AI Timeline" (2 min Remotion animation)
- **Key Points**:
  - AI development is accelerating, not slowing
  - This is different from previous technological shifts
  - The window for adaptation is closing
- **Checkpoint Quiz**: 3 questions on AI adoption trends
- **Media**: 3D visualization of computing power over time

#### 1.2: What AI Can Actually Do (2026 Edition)
- **Concept**: Demystifying AI capabilities vs hype
- **Video**: "AI Reality Check" (3 min)
- **Key Points**:
  - Language models: reasoning, coding, writing
  - Multi-modal: images, audio, video
  - Agentic systems: planning and execution
  - Current limitations and failure modes
- **Interactive**: Try Claude Desktop live demo
- **Checkpoint**: Identify 5 tasks AI can do for you

#### 1.3: Your Personal AI Assessment
- **Concept**: Baseline evaluation of student's AI exposure
- **Interactive Assessment**:
  - Current AI tools used (0-10 scale)
  - Comfort level with AI technology
  - Industry-specific AI exposure
  - Personal AI goals
- **AI Tutor Analysis**: Generates personalized learning path
- **Output**: Saved in `SessionState.SkillLevel` and `SessionState.Interests`

## Module 2: AI Literacy

**Goal**: Build foundational understanding of how AI works.

### Lessons

#### 2.1: How LLMs Actually Work
- **Concept**: Transformer architecture explained simply
- **Video**: "Inside the Black Box" (4 min)
- **Key Points**:
  - Tokens and context windows
  - Training vs inference
  - Temperature, top-p, and why responses vary
  - Prompt patterns that work
- **Interactive 3D**: Visualization of attention mechanism
- **Checkpoint**: Explain tokens in your own words

#### 2.2: The AI Tool Landscape
- **Concept**: Map the current AI ecosystem
- **Video**: "Tools of the Trade" (3 min)
- **Categories Covered**:
  - Text/Code: Claude, ChatGPT, Copilot
  - Image: Midjourney, DALL-E, Stable Diffusion
  - Audio: ElevenLabs, Suno
  - Video: Runway, Pika
  - Agents: AutoGPT, custom implementations
- **Interactive**: Tool selector board (tldraw)
- **Checkpoint**: Select tools relevant to your goals

#### 2.3: Prompt Engineering Fundamentals
- **Concept**: Communication patterns for AI
- **Video**: "Talking to Machines" (5 min)
- **Key Patterns**:
  - Role setting: "You are an expert in..."
  - Context provision: Background matters
  - Output formatting: JSON, markdown, code blocks
  - Chain of thought: "Think step by step"
  - Few-shot examples: Show, don't just tell
- **Interactive Exercises**:
  - Transform bad prompts → good prompts
  - AI tutor provides real-time feedback
- **Checkpoint**: Write 5 effective prompts for your use case

## Data Structures

### Course Module
```go
type CourseModule struct {
    ID          string       `json:"id"`
    Order       int          `json:"order"`
    Title       string       `json:"title"`
    Description string       `json:"description"`
    Duration    string       `json:"duration"` // "2 weeks"
    Lessons     []Lesson     `json:"lessons"`
}

type Lesson struct {
    ID           string           `json:"id"`
    ModuleID     string           `json:"module_id"`
    Order        int              `json:"order"`
    Title        string           `json:"title"`
    Description  string           `json:"description"`
    Duration     string           `json:"duration"`

    // Content types
    Video        *VideoContent    `json:"video,omitempty"`
    Text         *TextContent     `json:"text,omitempty"`
    Interactive  *InteractiveContent `json:"interactive,omitempty"`

    // Assessment
    Checkpoint   *Checkpoint      `json:"checkpoint,omitempty"`

    // Metadata
    Concepts     []string         `json:"concepts"`
    Difficulty   string           `json:"difficulty"` // "beginner" | "intermediate" | "advanced"
}

type VideoContent struct {
    Source    string `json:"source"`    // Remotion composition ID
    Duration  string `json:"duration"`  // "3:45"
    Thumbnail string `json:"thumbnail"`
}

type InteractiveContent struct {
    Type   string                 `json:"type"`   // "3d" | "board" | "quiz" | "code"
    Data   map[string]interface{} `json:"data"`
}

type Checkpoint struct {
    Type       string   `json:"type"`       // "quiz" | "exercise" | "project"
    Questions  []Question `json:"questions,omitempty"`
    Criteria   string   `json:"criteria,omitempty"`   // For exercises/projects
    PassScore  float64  `json:"pass_score"`           // 0.8 = 80%
}
```

### Student Progress Tracking
```go
type StudentProgress struct {
    StudentID      string            `json:"student_id"`
    TenantID       string            `json:"tenant_id"`

    Interests      []string          `json:"interests"`
    Goals          []string          `json:"goals"`

}
```

## AI Tutor Prompts by Module

### System Prompt Template
```text
You are an AI tutor for the "Get Ahead of AI" course. Your role is to:

1. Guide students through course material adaptively
2. Assess understanding through natural conversation
3. Provide personalized explanations based on skill level
4. Recommend media (videos, 3D visualizations, interactive boards)
5. Track progress and identify stuck points
6. Encourage hands-on experimentation

Student Context:
- Name: {UserName}
- Skill Level: {SkillLevel}
- Progress: {Progress}%
- Current Module: {CurrentModule}
- Interests: {Interests}
- Goals: {Goals}

Tutoring Principles:
- Be concise but thorough
- Use examples from student's domain/interests
- Suggest media when concepts would benefit from visualization
- Check understanding before moving forward
- Celebrate breakthroughs
- Normalize confusion - this is hard stuff
```

## Media Injection Triggers

The AI tutor should inject media based on conversation context:

| Trigger | Media Type | Condition |
|---------|-----------|-----------|
| Student asks "how does it work?" | 3D visualization | Concept is spatial/structural |
| Student seems confused | Interactive whiteboard | Concept benefits from diagramming |
| Student wants to "see" | Video | Full lesson video exists |
| Student says "show me example" | Code block | Live code demo |
| Student stuck on workflow | 2D board | Workflow mapping exercise |
| Student completes checkpoint | Celebration animation | Score > 80% |
| Student expresses frustration | Encouragement message | Tone analysis |
